# PulsoScena

"Descubrí lo que pasa cerca de ti"

Red social simplificada de eventos culturales desarrollada como proyecto final del Módulo 1 del Bootcamp BIT.

## Descripción

PulsoScena es una plataforma interactiva que permite a los usuarios explorar, publicar y compartir eventos culturales como conciertos, exposiciones de arte, festivales de teatro y más.

## Funcionalidades

- Inicio de sesión con validaciones
- Feed de publicaciones estilo red social
- Exploración de eventos culturales
- Sistema de me gusta
- Comentarios (agregar, editar y eliminar)
- Perfil de usuario con estadísticas
- Crear y publicar nuevos eventos
- Diseño responsive con menú hamburguesa

## Tecnologías

- HTML5 semántico
- CSS3
- Bootstrap 5.3
- JavaScript Vanilla
